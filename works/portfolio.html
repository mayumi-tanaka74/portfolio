ポートフォリオサイト

制作箇所 デザイン・コーディング
使用ツール Illustrator / Photoshop / HTML / CSS / jQuery
期間 デザイン：3週間 コーディング：3週間

TOPページは、昼・夜で空と月の色を変化させることで、時代やお客様に合ったデザインを心がけることを表現しました。空色をイメージした淡いカラーを全体的に使用することで柔らかい印象をもってもらえるよう制作しました。
